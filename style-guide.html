01 — Voice

Direct, technical, custody-first.

We describe what the product does and constrains, not what it "empowers." Confident without hype; regulated-industry literate; the through-line is always custody — what stays inside the customer's boundary.

✓ Do
  • "Attacks ride the same rails; the fabric is where they stop."
  • Name the real frameworks — HIPAA, ITAR/CMMC, the CLOUD Act, OCAP.
  • "One builder, one phone call, real answers — no sales pitch."
  • Plain declaratives; sentence case; lead with what stays in the customer's control.
✕ Don't
  • "Revolutionary," "cutting-edge," "empower," "unlock."
  • Exclamation marks or vague benefit claims.
  • Anything implying data leaves the boundary.
  • All-caps outside the wordmark.

07 — Process How-To's

The procedures, so nothing is re-derived.

The repeatable methods behind the brand — copy-pasteable. Full detail lives in BRAND.md; the essentials are here.

A Asset production

A1 · Icon → transparent → webp

Solid-bg PNG → transparent via luminance-keyed alpha (keeps white linework crisp), then lossless webp.

# black bg → transparent, force pure-white RGB
convert in.png \( +clone -colorspace Gray \) -alpha off \
  -compose CopyOpacity -composite \
  -channel RGB -fill white -colorize 100 +channel PNG32:t.png
cwebp -quiet -lossless -alpha_q 100 t.png -o images/name-icon.webp

A2 · Video for web (.au-infographic)

H.264, web-safe pixel format, faststart, no audio — plus a webp poster from the last frame.

ffmpeg -y -framerate 24 -i frames/f_%04d.png \
  -c:v libx264 -pix_fmt yuv420p -crf 20 -preset slow \
  -movflags +faststart -an video/name.mp4
ffmpeg -y -sseof -0.3 -i video/name.mp4 -vframes 1 \
  -c:v libwebp -quality 82 images/name-poster.webp

A3 · Watermark removal

Preferred: full-frame masked inpaint — locate the static glyph via the temporal-minimum across frames, then per-frame cv2.INPAINT_TELEA. Corner-crop only when nothing important lives in the corner. To brand a clip, see VIDEO_BRANDING_WORKFLOW.md.

A4 · Infographic translucency

Soften flat-white embedded graphics so the dark ground shows through; full opacity on hover.

.woven-split-media img{ opacity:.78; transition:opacity .3s }
.woven-split-media img:hover{ opacity:1 }

A6 · GIF builds — tools/gif_process.py

Float a light-bg GIF onto the dark theme (luminance→alpha), recolor (white·copper·gray·keep·fire), protect a central emblem via --radial/--accent, export 8-bit-alpha WebP (preferred) or 1-bit GIF.

python tools/gif_process.py in.gif images/name.webp \
  --color white --scale 0.5            # --format gif for 1-bit

C Build, delivery & deploy

C10 · CSS change delivery — the gotcha

Pages load style.min.css and there is no build step. Every change goes in both style.css and style.min.css, then bump the cache key everywhere.

grep -rl 'style.min.css?v=2' --include=*.html . \
  | xargs sed -i 's#style.min.css?v=2#style.min.css?v=3#g'

C11 · Deploy & verify

Push to main → the serialized pages.yml workflow. Watch the run to green; don't push and hope. Note: this session's egress blocks fetching islandmountain.io — verify via the GitHub API/tree instead.

D Verification / QA

D13 · HTML integrity scan

After editing HTML: no NUL/truncation, terminator present, balanced sections. Two traps: grep -c $'\\x00' collapses to an empty match; grep -c exits non-zero on zero hits.

LC_ALL=C grep -qP '\x00' f.html && echo "NUL FOUND"   # correct form
tail -c 20 f.html | grep -o '</html>'
grep -oE '<section' f.html | wc -l   # vs </section>

D14 · Render check

Drive the preinstalled Chromium — absolute import, 390px mobile / 1000px+ desktop, screenshot el.closest('section'), and measure computed styles rather than eyeballing.
