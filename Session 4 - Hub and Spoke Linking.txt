HANDOFF.md has the full site inventory, HTML patterns, and known issues. Read it first.

- The five vertical pages link to solutions.html ONLY through global nav and footer (every page has these -- they don't signal hub-spoke relationship).

- No vertical page has contextual body links back to solutions.html.

Adding links from each testimonial cite to the appropriate vertical (through solutions.html or directly) would strengthen the hub-spoke signal.

JOB 4 -- ADD BreadcrumbList SCHEMA TO VERTICALS (5 files, all over 400 lines, use sed):
Each vertical page already has JSON-LD schema in the head. Add a BreadcrumbList schema to each vertical page to match the visual breadcrumb added in Job 1. This tells search engines explicitly about the hub-spoke hierarchy.

Format:
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [

  ]
}

Add this as a second <script type="application/ld+json"> block in the <head> of each vertical page, after the existing FAQPage schema.
