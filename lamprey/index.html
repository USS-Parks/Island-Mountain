The Lamprey MAI stack

Twelve layers from hardware foundation up through the operator dashboard. Every layer ships in this installer, runs locally on your machine, and stays inside your network boundary.

System requirements

  • Windows 10 or 11, 64-bit
  • Administrator rights for first install
  • ~30 MB free disk for the application
  • Hardware AI inference is optional — the launcher and compliance demos run on any CPU
  • No network connection required

For GPU-accelerated inference, see the Technology page.

What to expect when you double-click the installer

1
“Windows protected your PC” (blue screen)

SmartScreen warns about unrecognized publishers. Click More info, then Run anyway. We’re working on Authenticode code-signing; until that lands, this warning is expected for every fresh build.

2
User Account Control (UAC) prompt

Windows asks for administrator rights because the installer writes into Program Files. Click Yes. Lamprey MAI itself runs as your normal user once installed.

3
Setup wizard with the gold Lamprey MAI splash

Step through the wizard. On the last page, make sure Create a desktop icon is checked. Optionally leave Launch Lamprey MAI ticked to start it immediately after install.

4
Double-click the desktop icon

A brief Win32 splash with the Lamprey silhouette appears for ~4 seconds, then a terminal opens with the ASCII Lamprey banner and an interactive mai> prompt.

5
Try it

At the mai> prompt, type demo to see the six narrated compliance scenarios (HIPAA, ITAR, OCAP, multi-domain, tamper-evident audit, trust manifold) walk through their phases. Or type start to launch the headless inference daemon, status to check the install, help for the full command list.

What you’re installing

Three executables under C:\Program Files\Lamprey MAI\:

  • lamprey-mai.exe — the front-door launcher with terminal UI
  • lamprey-mai-api.exe — headless inference + compliance governance daemon (REST + gRPC)
  • lamprey-mai-admin.exe — operator CLI: demos, audit, backup/restore, ops

No background services are registered. No scheduled tasks. No system-tray app. Nothing runs unless you start it.

Data handling: Lamprey MAI is designed to operate fully air-gapped. The installer makes no network requests after install. The application does not send telemetry, crash reports, or usage data anywhere. Compliance audit logs, model outputs, and configuration all stay on the local machine. Frequently asked questions

Why does Windows warn me when I run this?

The installer is not yet Authenticode code-signed. SmartScreen flags any binary from an “unrecognized publisher” until enough installs of the same SHA256 accumulate. We’re in the process of getting an EV (Extended Validation) code-signing certificate; once it’s issued, the warning will be gone immediately on future builds. In the meantime, the SHA256 above is the primary trust signal.

Can I send this installer over email?

No. Gmail, Outlook, and most corporate mail gateways block .exe attachments outright (even inside .zip). Send your recipient the link to this page instead — https://islandmountain.io/lamprey/ — and they can download it and verify it themselves.

Does Lamprey MAI need an internet connection?

No. Everything runs locally on your hardware. The installer itself does not contact any server during install. The application does not phone home, check for updates, or transmit data after install. Air-gapping is a design constraint, not an opt-in.

How do I uninstall it?

Settings → Apps → Installed apps → search “Lamprey MAI” → Uninstall. Or run the uninstaller directly at C:\Program Files\Lamprey MAI\unins000.exe. The uninstaller removes everything under Program Files\Lamprey MAI and the desktop/Start Menu shortcuts.

Where can I file issues or ask questions?

Email basho@islandmountain.io or call 1-801-609-1130. Include the version number (1.1.1) and the first few lines from status at the mai> prompt.

What about Linux or macOS?

Linux support ships with the Landfall hardware appliances. macOS is not currently a primary target. For server / appliance deployment, see Technology and Summit.
